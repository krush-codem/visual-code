**Visual-Code IDE** is a high-performance, logical engineering studio. It bridges the gap between raw source code and architectural reality by providing real-time, interactive visualization for modern engineering workflows.

> "Stop reading code. Start seeing it."

---

## 🚀 The Dual-Engine Core

### 🔷 ReactArchitect (Project Engine)
*Designed for Frontend Architects.* Map out your component hierarchy and data flow with zero configuration.
- **Component Blueprints:** Instant identification of functional components.
- **Usage Archetypes:** Visual "usage" lines link tags back to their definitions.
- **Hook Intelligence:** Dedicated nodes for `useState`, `useEffect`, and custom hooks.
- **Architecture Sync:** The graph evolves in real-time as you edit your JSX/TSX.

### 🐍 PyViz (Logic Engine)
*Designed for Algorithm Engineers.* Perfect for mastering DSA and deep logic analysis.
- **Logical Sequencing:** Every execution step is numbered (#1, #2, ...) for flow tracking.
- **Recursion Detection:** Specialized red highlights and call-stack warnings for recursive logic.
- **Object Formation:** Visually link instances (Trees, Heaps) back to their Class Archetypes.
- **Deep AST Mapping:** Peek under the hood at the raw Python Abstract Syntax Tree.

## 🛠️ High-Performance Tech Stack

- **Core:** [React 19](https://react.dev/) & [Vite](https://vitejs.dev/)
- **Visuals:** [@xyflow/react](https://reactflow.dev/) (React Flow)
- **Parsing:** [@babel/parser](https://babeljs.io/) & [Pyodide](https://pyodide.org/)
- **UI:** [Tailwind CSS](https://tailwindcss.com/) & [Lucide Icons](https://lucide.dev/)
- **Storage:** [IndexedDB](https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API) (via idb)

## 🏁 Getting Started

### 1. Installation
```bash
git clone https://github.com/your-username/visual-code-ide.git
cd visual-code-ide
npm install
```

### 2. Launch
```bash
npm run dev
```

### 3. Workflow
- **For React:** Navigate to `/project` and click **OPEN PROJECT**.
- **For Python:** Navigate to `/live` and start typing your algorithm.

## 📈 Advanced Features

- **Direct Source Mapping:** Nodes display the actual line of code they represent.
- **Insight Panel:** Slide-out technical breakdown with technical data and execution roles.
- **Mobile Responsive:** A fully tabbed mobile interface for logic analysis on the go.
- **Zero-Empty-Nodes:** Hardened walkers ensure every piece of logic is visually mapped.
